## Temas de la Semana

- **Arrays**: índices, `.push()`, `.pop()`, `.shift()`, `.unshift()`
- **LIFO y FIFO**: Pilas (tubo de Pringles) y Colas (fila del súper)
- **Objetos**: propiedades, acceso con dot notation, mutación de estado
- **Métodos de array**: `.map()` (transformar), `.filter()` (filtrar), `.reduce()` (acumular)
- **El DOM**: qué es el árbol de nodos, cómo el navegador convierte HTML en objetos
- **Selección de elementos**: `document.querySelector()`, `document.querySelectorAll()`
- **Manipulación del DOM**: `.innerHTML`, `.textContent`, `.appendChild()`, `.createElement()`
- **Eventos**: `addEventListener()`, el objeto `event`, tipos de eventos (`click`, `input`, `keydown`)
- **Renderizado dinámico**: generar HTML desde un array de datos con `.map()` + `.join('')`
- **Patrón Buscador**: capturar input → `.filter()` → volver a renderizar
